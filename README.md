# smart-municipality (login)

## Run locally (XAMPP)

1. Copy configuration:
   - Copy `Modules/config.example.php` to `Modules/config.php`
2. Edit `Modules/config.php` with your MySQL credentials and database name.
3. Put this project under your XAMPP `htdocs` and open:
   - `http://localhost/page_login/index.php?route=login`

## Database

- Import `Modules/schema.sql` into your MySQL database.
